Projects

TECHshop

Full stack web application, using React for front end, Redux for state management, Django for backend and API. PostgreSQL for database. Project uses sign up/login functionality with token based-authentication. Product data is stored in PostgreSQL relational database and uses API calls to populate the correct product details for each of the products in the product details React component. The website includes a wishlist page which serves as a favorites for products to buy in the future and also includes a shopping cart with a payment gateway integration. This project is conceptual and represents a demonstration, not an actual contracted piece of work for a client. This effort was put together by a small team of developers. I was responsible for working on both the front and back end of this demo e-commerce platform.

Todo List

Multi-page Todo List client side web application, developed with React. Redux was used for state management and filter functionality. Create/delete todo items, see more details, or filter, based on completion.

This is a prototype with limited functionality. Currently, there is no use of local or session storage to preserve the list, so when the page is closed, it will always reload. I may make this into a full stack application with login functionality and store todos in a database for each user, but the intent of this project was to create a semi-functional todo list web application, totally using front end development technology and not relying on a database or an API. The prototype is 100% finished, but this project, in its current form, is about 70% complete.
